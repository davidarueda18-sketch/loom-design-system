# Loom Design System

Sistema de diseño de componentes construido con **React**, **TypeScript** y **Vanilla Extract**.

## Estructura

| Categoría | Descripción |
|---|---|
| **Tokens** | Variables de diseño: color, spacing, tipografía, sombras |
| **Primitives** | Bloques base: Box, Stack, Inline |
| **Components** | Componentes con estado: Dialog, Tabs, Accordion |
| **Patterns** | Composiciones de alto nivel: FormField, PageLayout |

## Primitivos de layout

Los tres primitivos de layout son la base de toda composición:

- **Box** — contenedor genérico con control de padding basado en tokens
- **Stack** — apila elementos verticalmente con gap uniforme
- **Inline** — alinea elementos horizontalmente con control de gap, align y justify
